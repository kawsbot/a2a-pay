# A2A Pay
Agent-to-Agent Payment Protocol on Solana.
## Structure
- programs/registry/ - Anchor program for service registration (Rust)
- programs/escrow/ - Anchor program for escrow payments (Rust)
- sdk/ - TypeScript SDK with A2AClient class
- cli/ - CLI tool for testing
- demo/ - Demo agent scripts
